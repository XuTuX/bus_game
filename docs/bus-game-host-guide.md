# 버스 게임 운영자 진행북

> 운영자가 시간 관리, 방 이동 안내, 점수 기록, 멘트를 바로 진행할 수 있도록 만든 문서

## 2. 운영자용 전체 진행 체크리스트

## 시작 10분 전

- [ ] 전체 맵을 중앙 진행 공간에 붙인다.
- [ ] 1번 버스 방, 2번 버스 방, 지하철 방 표지를 붙인다.
- [ ] 각 방에 방별 미니 맵과 행동지를 둔다.
- [ ] 버스 말 2개와 지하철 말 1개를 시작 위치에 둔다.
- [ ] 점수 기록지를 운영자 자리 가까이에 둔다.
- [ ] 타이머를 18분 라운드용으로 맞춘다.
- [ ] 명찰에 P1~P10과 팀 색상을 적는다.
- [ ] 라운드별 방 배치표를 출력해 각 방 앞에 붙인다.

## 시작 직후

- [ ] 플레이어 출석을 확인한다.
- [ ] P번호와 팀 색상을 확인시킨다.
- [ ] 게임 목표를 설명한다.
- [ ] 3개의 방이 동시에 다른 교통수단을 조작한다고 설명한다.
- [ ] 점수는 라운드마다 기록하고 5라운드 후 합산한다고 설명한다.

## 튜토리얼

- [ ] 1번 버스가 3칸 이동하는 예시를 보여준다.
- [ ] 2번 버스가 과속으로 5칸 이동하는 예시를 보여준다.
- [ ] 두 버스가 거리 1칸이면 감점된다고 설명한다.
- [ ] 충돌은 `-6점`이라고 강조한다.
- [ ] 지하철은 Snake 노선만 따라간다고 설명한다.
- [ ] 환승역 신호 예시를 1번 보여준다.
- [ ] 행동지 작성 예시를 보여준다.

## 각 라운드 시작 전

- [ ] 이번 라운드 방 배치를 읽는다.
- [ ] 모든 플레이어가 자기 방으로 이동했는지 확인한다.
- [ ] 각 방에 행동지가 있는지 확인한다.
- [ ] 타이머 18분을 시작한다.
- [ ] 10분 남았을 때 중간 안내를 한다.
- [ ] 3분 남았을 때 제출 준비 안내를 한다.

## 각 라운드 종료 후

- [ ] 세 방의 행동지를 모두 수거한다.
- [ ] 1번 버스 이동을 전체 맵에 반영한다.
- [ ] 2번 버스 이동을 전체 맵에 반영한다.
- [ ] 지하철 이동을 전체 맵에 반영한다.
- [ ] 목적지, 색상 미션, 회색 공용 구역 점수를 기록한다.
- [ ] 버스 근접 또는 충돌 여부를 확인한다.
- [ ] 지하철 환승 신호 효과를 적용한다.
- [ ] 라운드 점수와 누적 점수를 공개 점수판에 적는다.
- [ ] 다음 라운드 방 배치를 안내한다.

## 중간 휴식

- [ ] 현재 누적 점수를 다시 계산한다.
- [ ] 행동지 누락 여부를 확인한다.
- [ ] 버스와 지하철 현재 위치를 다시 확인한다.
- [ ] 남은 라운드 수와 재개 시간을 안내한다.

## 최종 집계

- [ ] 1~5라운드 점수가 모두 기록되었는지 확인한다.
- [ ] 감점이 반영되었는지 확인한다.
- [ ] 동점 팀 또는 플레이어가 있는지 확인한다.
- [ ] 동점 처리 기준을 적용한다.
- [ ] 우승 팀 또는 우승 플레이어를 확정한다.

## 종료 후 정리

- [ ] 점수 기록지를 사진으로 남긴다.
- [ ] 행동지를 라운드 순서대로 모은다.
- [ ] 맵, 말, 카드, 펜을 회수한다.
- [ ] 분실물과 쓰레기를 확인한다.

## 3. 운영자 진행 멘트

### 게임 시작 전 안내

안녕하세요. 지금부터 약 2시간 30분 동안 **버스 게임**을 진행하겠습니다.  
오늘은 10명이 3개의 방으로 나뉘어서 1번 버스, 2번 버스, 지하철을 움직입니다. 각 방의 선택은 따로 이루어지지만, 결과는 하나의 전체 맵에 합쳐지고 최종 점수에 모두 반영됩니다.

명찰에 적힌 P번호와 팀 색상을 먼저 확인해 주세요. 라운드가 끝날 때마다 제가 다음 방 배치를 안내드릴 예정입니다.

### 룰 설명 멘트

이 게임은 총 5라운드입니다. 각 라운드마다 여러분은 1번 버스 방, 2번 버스 방, 지하철 방 중 하나로 이동합니다.

1번 버스는 안정적으로 목적지와 미션을 달성하는 역할입니다.  
2번 버스는 더 빠르게 움직일 수 있지만, 충돌과 근접 감점 위험이 큽니다.  
지하철은 정해진 노선을 따라 움직이며, 환승역을 지나면 버스의 이동 결과에 영향을 줄 수 있습니다.

점수는 목적지 도착, 미션 성공, 회색 공용 구역 점령, 지하철 역 도착 등으로 얻습니다. 반대로 버스끼리 너무 가까워지거나 충돌하면 감점됩니다.

가장 중요한 건 한 방만 잘한다고 이기는 게임이 아니라는 점입니다. 내가 있는 방의 선택이 다른 방을 도울 수도 있고 방해할 수도 있습니다.

### 라운드 시작 멘트

이제 `__라운드`를 시작하겠습니다.  
배치표를 확인하고 각자 지정된 방으로 이동해 주세요. 이동이 끝나면 방 안에서 현재 맵과 점수판을 확인하고, 이번 라운드 행동을 정해 행동지에 적어 주세요.

제한 시간은 18분입니다. 3분 남았을 때 한 번 더 안내드리겠습니다.

### 라운드 중간 멘트

10분 남았습니다. 아직 이동 방향을 정하지 않은 방은 먼저 이동을 확정해 주세요.  
행동지는 라운드 종료 전에 운영자에게 제출해야 점수가 인정됩니다.

### 라운드 종료 3분 전 멘트

3분 남았습니다. 이제 최종 행동을 정하고 행동지를 작성해 주세요.  
이동 칸, 도착 위치, 추가 행동이 빠지지 않았는지 확인해 주세요.

### 라운드 종료 멘트

라운드 종료입니다. 모든 방은 행동지를 제출해 주세요.  
지금부터 운영자가 전체 맵에 이동 결과를 반영하고 점수를 계산하겠습니다. 잠시만 기다려 주세요.

### 방 이동 안내 멘트

다음은 `__라운드`입니다.  
1번 버스 방은 `____`, 2번 버스 방은 `____`, 지하철 방은 `____`입니다.  
3분 안에 다음 방으로 이동해 주세요. 이전 방의 행동지는 그대로 두고, 개인 물품만 챙겨 이동하면 됩니다.

### 중간 휴식 안내

2라운드까지 종료되었습니다. 지금부터 8분간 휴식입니다.  
현재 누적 점수는 점수판에 업데이트해 두겠습니다. 휴식 후에는 3라운드부터 바로 시작하니, 재개 시간에 맞춰 자리로 돌아와 주세요.

### 최종 점수 발표 멘트

5라운드가 모두 종료되었습니다. 지금부터 최종 점수를 집계하겠습니다.  
목적지 점수, 미션 점수, 지하철 점수, 충돌 감점을 모두 합산한 뒤 최종 순위를 발표하겠습니다.

### 우승자 발표 멘트

최종 점수 집계가 끝났습니다.  
오늘의 우승은 총 `__점`을 기록한 `____`입니다. 축하드립니다.  
동점이 있는 경우에는 충돌 감점, 목적지 도착 횟수, 지하철 지정 역 도착 횟수 순서로 순위를 정했습니다.

오늘 게임은 여기까지입니다. 사용한 카드와 펜은 방 안에 두고, 명찰은 나가실 때 반납해 주세요.

## 8. 운영자 판정 순서

라운드 종료 후 아래 순서대로 처리한다.

1. 행동지 3장을 모두 수거한다.
2. 1번 버스 이동을 맵에 표시한다.
3. 2번 버스 이동을 맵에 표시한다.
4. 지하철 이동을 노선표에 표시한다.
5. 버스 목적지 도착 여부를 확인한다.
6. 색상 카드 또는 미션 성공 여부를 확인한다.
7. 지하철 지정 역, 환승역, 연속 이동 보너스를 확인한다.
8. 두 버스의 최종 거리를 확인한다.
9. 충돌 또는 경로 교차가 있었는지 확인한다.
10. 회색 공용 구역 점수를 확인한다.
11. 지하철 환승 신호를 적용한다.
12. 라운드 점수와 누적 점수를 기록한다.
13. 다음 라운드 배치를 안내한다.

## 10. 진행 중 실수 방지 메모

- [ ] 라운드 시작 전 반드시 현재 버스와 지하철 위치를 모두에게 보여준다.
- [ ] 세 방의 행동지는 동시에 제출받는다.
- [ ] 행동지를 받은 뒤에는 수정할 수 없게 한다.
- [ ] 충돌 판정은 최종 위치뿐 아니라 이동 경로도 확인한다.
- [ ] 지하철 신호는 한 라운드에 1번만 적용한다.
- [ ] 점수판에 적기 전에 방별 점수 합계를 한 번 소리 내어 확인한다.
- [ ] 플레이어가 룰을 물어보면 새 규칙을 만들지 말고 룰북의 빠른 판정표를 따른다.

## 11. 운영 난이도를 낮추기 위한 추천 수정안

1. 첫 행사에서는 라운드별 보너스 카드를 사용하지 않고 기본 점수만 사용한다.
2. 미션 카드는 팀별 1장씩만 공개 미션으로 둔다.
3. 충돌 판정이 어렵다면 경로 교차 충돌을 빼고 같은 칸 도착만 충돌로 본다.
4. 지하철 신호는 지연 신호만 사용하고 환승 지원은 제외한다.
5. 최종 점수 집계 시간을 줄이기 위해 라운드 종료마다 누적 점수를 바로 공개한다.
