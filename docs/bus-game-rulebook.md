# 버스 게임 룰북

> 10명 / 5라운드 / 3개 방 / 총 2시간 30분 진행용

---

## 1. 게임 개요

**버스 게임**은 10명의 플레이어가 3개의 방으로 나뉘어 5라운드 동안 교통수단을 조작하고, 각 방의 선택 결과를 합산해 우승 팀 또는 우승 플레이어를 정하는 전략형 오프라인 팀 게임이다.

### 왜 3개의 방으로 나누는가

10명이 한 공간에서 동시에 한 판을 조작하면 대기 시간이 길고, 발언권이 특정 사람에게 몰리기 쉽다. 이 게임은 10명을 `3명 / 3명 / 4명`으로 나누어 각 방에서 동시에 결정을 내리게 한다.

- 한 방의 인원이 적어져 모두가 의견을 낼 수 있다.
- 3개의 교통수단이 동시에 움직여 라운드마다 전체 판이 크게 변한다.
- 내가 없는 방의 선택도 내 점수에 영향을 주므로, 방 배치와 협상이 중요해진다.

### 3개 방의 역할

| 방 | 담당 교통수단 | 성격 | 핵심 역할 |
| --- | --- | --- | --- |
| 1번 버스 방 | 1번 버스 | 안정형 | 목적지 도착, 색상 미션 성공, 중앙 구역 점령 |
| 2번 버스 방 | 2번 버스 | 공격형 | 1번 버스 견제, 위험 구역 선점, 고득점 리스크 선택 |
| 지하철 방 | 지하철 | 노선형 | 정해진 노선을 따라 연속 이동, 역 도착 보너스, 버스 이동 방해 또는 지원 |

이 게임의 핵심 재미는 **각 방의 선택과 이동 결과가 전체 점수에 영향을 주는 전략형 오프라인 팀 게임**이라는 점이다. 같은 라운드 안에서도 1번 버스, 2번 버스, 지하철이 서로의 이동을 돕거나 방해하며, 5라운드 동안 쌓인 판단이 최종 순위를 만든다.

---

## 2. 기본 구성

### 추천 팀 구성

기본 진행은 **5개 팀 2명**을 추천한다.

| 팀 | 플레이어 |
| --- | --- |
| 빨강 팀 | P1, P2 |
| 파랑 팀 | P3, P4 |
| 초록 팀 | P5, P6 |
| 노랑 팀 | P7, P8 |
| 보라 팀 | P9, P10 |

개인전으로 진행할 경우 팀 점수 대신 P1~P10 개인 점수판을 사용한다. 룰은 동일하며, 점수는 해당 방에서 얻은 점수를 그 방에 있던 개인에게 나누지 않고 각각 기록한다.

### 권장 맵

운영 난이도를 낮추기 위해 전체 맵은 `7 x 7` 격자판을 추천한다.

- 각 칸에는 색상, 목적지, 역, 회색 공용 구역 중 하나의 표시를 둔다.
- 중앙 `3 x 3` 범위 중 4칸을 **회색 공용 구역**으로 지정한다.
- 1번 버스와 2번 버스는 도로 칸을 이동한다.
- 지하철은 정해진 역 순서만 따라 이동한다.

### 회색 공용 구역

회색 공용 구역은 모든 방이 관심을 가지는 특수 칸이다.

- 버스가 라운드 종료 시 회색 칸에 있으면 `+2점`
- 지하철이 회색 구역과 연결된 환승역을 지나가면 버스에 신호 효과를 줄 수 있다.
- 두 버스가 회색 구역 안에서 너무 가까워지면 감점 위험이 커진다.

## 3. 플레이어 배치 방식

각 라운드마다 방별 인원은 `3명 / 3명 / 4명`을 기본으로 한다.

운영자는 아래 배치표를 그대로 읽고 안내하면 된다. 이 배치는 모든 플레이어가 5라운드 동안 지하철을 2회, 버스를 3회 경험하도록 설계되어 있다.

| 라운드 | 1번 버스 방 3명 | 2번 버스 방 3명 | 지하철 방 4명 |
| --- | --- | --- | --- |
| 1라운드 | P5, P6, P7 | P8, P9, P10 | P1, P2, P3, P4 |
| 2라운드 | P1, P3, P9 | P2, P4, P10 | P5, P6, P7, P8 |
| 3라운드 | P2, P4, P8 | P3, P6, P7 | P1, P5, P9, P10 |
| 4라운드 | P3, P5, P7 | P1, P4, P8 | P2, P6, P9, P10 |
| 5라운드 | P2, P6, P10 | P1, P5, P9 | P3, P4, P7, P8 |

### 배치 운영 원칙

- 각 방 입구에 라운드별 배치표를 붙인다.
- 플레이어 명찰에는 `P번호`와 `팀 색상`을 크게 적는다.
- 라운드가 끝나면 운영자가 다음 라운드 방 배치를 먼저 읽고, 이동 시간을 준다.
- 같은 사람이 계속 같은 방에 들어가지 않도록 운영자가 임의 변경하지 않는다.

---

## 4. 라운드 진행 방식

각 라운드는 약 18분으로 진행한다.

1. 운영자가 라운드 시작을 알린다.
2. 플레이어는 배정된 방으로 이동한다.
3. 각 방은 2분 동안 현재 맵, 점수판, 이번 라운드 미션을 확인한다.
4. 각 방은 8분 동안 이동 계획과 행동을 논의한다.
5. 각 방은 5분 안에 최종 행동지를 작성해 운영자에게 제출한다.
6. 운영자는 세 방의 행동지를 모두 모은다.
7. 운영자는 1번 버스, 2번 버스, 지하철 순서로 이동 결과를 전체 맵에 반영한다.
8. 운영자는 방 사이 상호작용, 보너스, 감점을 계산한다.
9. 운영자는 점수판에 라운드 점수를 기록한다.
10. 운영자는 다음 라운드 방 배치를 안내한다.

### 라운드별 행동지에 적을 내용

| 항목 | 적는 내용 |
| --- | --- |
| 방 이름 | 1번 버스 / 2번 버스 / 지하철 |
| 라운드 | 1~5라운드 |
| 대표 작성자 | 방 안 플레이어 중 1명 |
| 이동 선택 | 이동 칸 수, 방향, 도착 칸 또는 도착 역 |
| 추가 행동 | 미션 수행, 신호 사용, 견제, 패스 등 |
| 목표 | 도착 / 색상 / 중앙 구역 / 방해 / 회피 중 선택 |

---

## 5. 각 방의 역할과 룰

## 5.1 1번 버스 방

### 이 방에서 하는 일

1번 버스 방은 1번 버스를 조작한다. 1번 버스는 안정적으로 점수를 쌓는 교통수단이다. 목적지 도착, 색상 카드 달성, 중앙 회색 칸 점령에 강하다.

### 기본 이동 규칙

- 1번 버스는 한 라운드에 최대 `4칸` 이동한다.
- 상하좌우로만 이동할 수 있고 대각선 이동은 불가능하다.
- 같은 칸을 한 라운드 안에서 두 번 지나갈 수 없다.
- 보드 밖으로 나갈 수 없다.
- 2번 버스가 있는 칸으로 들어갈 수 없다.

### 선택 가능한 행동 예시

| 행동 | 효과 |
| --- | --- |
| 안전 운행 | 최대 3칸 이동. 충돌 감점이 발생하지 않으면 추가 `+1점` |
| 빠른 운행 | 최대 4칸 이동. 목적지 도착 시 좋지만, 다른 버스와 가까워지면 감점 위험 증가 |
| 정류장 정차 | 인접한 목적지 또는 색상 칸에서 미션 확인. 미션 성공 시 `+3점` |
| 중앙 진입 | 회색 공용 구역으로 이동. 라운드 종료 시 해당 칸에 있으면 `+2점` |

### 성공 조건

- 이번 라운드 목적지 칸에 도착한다.
- 팀 색상 카드와 같은 색상 칸에 도착한다.
- 2번 버스와 충돌하지 않는다.
- 회색 공용 구역을 안정적으로 점령한다.

### 실패 조건

- 이동 경로가 보드 밖으로 나간다.
- 이미 다른 버스가 있는 칸으로 이동한다.
- 행동지를 시간 안에 제출하지 못한다.
- 충돌 또는 근접 감점이 발생한다.

### 점수 획득 방식

| 조건 | 점수 |
| --- | ---: |
| 목적지 도착 | +5 |
| 색상 카드 또는 미션 성공 | +3 |
| 회색 공용 구역 점령 | +2 |
| 안전 운행 보너스 | +1 |
| 2번 버스와 거리 1칸으로 종료 | -2 |
| 2번 버스와 충돌 | -6 |

### 주의사항

- 1번 버스는 안정형이지만, 중앙 회색 칸을 노리면 2번 버스와 충돌하기 쉽다.
- 2번 버스의 공격형 이동을 예측해야 한다.
- 지하철이 환승역을 지나가면 신호 지연을 받을 수 있다.

---

## 5.2 2번 버스 방

### 이 방에서 하는 일

2번 버스 방은 2번 버스를 조작한다. 2번 버스는 1번 버스와 비슷하게 도로를 이동하지만, 더 공격적이고 리스크가 큰 선택을 할 수 있다.

### 1번 버스와의 차별점

- 2번 버스는 한 라운드에 최대 `5칸` 이동할 수 있다.
- 대신 4칸 이상 이동하면 **과속 상태**가 된다.
- 과속 상태에서 1번 버스와 거리 1칸으로 끝나면 감점이 `-2점`이 아니라 `-3점`이 된다.
- 2번 버스는 1번 버스가 노리는 목적지나 회색 칸을 선점해 방해할 수 있다.

### 기본 이동 규칙

- 상하좌우로만 이동한다.
- 한 라운드에 최대 5칸 이동한다.
- 같은 칸을 한 라운드 안에서 두 번 지나갈 수 없다.
- 1번 버스가 있는 칸으로 들어갈 수 없다.
- 1번 버스와 이동 경로가 서로 교차하면 충돌로 처리한다.

### 선택 가능한 행동 예시

| 행동 | 효과 |
| --- | --- |
| 일반 운행 | 최대 3칸 이동. 안정적으로 미션 수행 |
| 과속 운행 | 4~5칸 이동. 목적지 도착 가능성이 높지만 감점 위험 증가 |
| 차선 방해 | 1번 버스의 예상 경로와 가까운 칸으로 이동. 성공 시 견제 보너스 가능 |
| 중앙 선점 | 회색 공용 구역에 먼저 도착해 `+2점` 획득 |

### 성공 조건

- 목적지 또는 색상 미션을 달성한다.
- 1번 버스가 회색 공용 구역 보너스를 얻지 못하게 만든다.
- 과속을 사용하고도 충돌이나 근접 감점을 피한다.

### 실패 조건

- 1번 버스와 충돌한다.
- 과속 후 거리 1칸으로 끝난다.
- 지하철 신호 지연을 받은 상태에서 무리하게 중앙을 노린다.

### 점수 획득 방식과 리스크

| 조건 | 점수 |
| --- | ---: |
| 목적지 도착 | +5 |
| 색상 카드 또는 미션 성공 | +3 |
| 회색 공용 구역 점령 | +2 |
| 1번 버스의 목적지 선점 | +2 |
| 과속 후 목적지 도착 | 추가 +1 |
| 1번 버스와 거리 1칸으로 종료 | -2 |
| 과속 후 거리 1칸으로 종료 | -3 |
| 1번 버스와 충돌 | -6 |

### 주의사항

- 2번 버스는 고득점 가능성이 높지만, 감점도 자주 발생한다.
- 방해를 노리더라도 자기 팀의 미션 점수를 놓치지 않는 것이 중요하다.
- 충돌은 양쪽 버스 모두 큰 감점을 받으므로, 확실하지 않으면 거리 2칸 이상을 유지한다.

---

## 5.3 지하철 방

### 이 방에서 하는 일

지하철 방은 지하철을 조작한다. 지하철은 버스처럼 자유롭게 움직이지 않고, 정해진 노선을 따라 움직인다. 대신 연속 이동, 역 도착 보너스, 환승역 효과로 전체 판에 큰 영향을 줄 수 있다.

### 지하철 이동 방식

지하철은 아래와 같은 **Snake 방식 노선**을 따른다.

```text
S1 -> S2 -> S3 -> S4
                  |
S8 <- S7 <- S6 <- S5
|
S9 -> S10 -> S11 -> S12
```

- 지하철은 현재 역에서 노선 순서대로만 이동한다.
- 한 라운드에 `1~4역` 이동할 수 있다.
- 뒤로 이동할 수 없다.
- S12에 도착한 뒤 추가 이동이 남으면 S1로 순환한다.
- S4, S8, S10은 환승역이다.

### 선택 가능한 행동 예시

| 행동 | 효과 |
| --- | --- |
| 정차 | 1역 이동. 원하는 역에 정확히 멈추기 쉽다 |
| 일반 운행 | 2~3역 이동. 안정적인 역 보너스 획득 |
| 급행 운행 | 4역 이동. 연속 이동 보너스를 노리지만 원하는 역을 지나칠 수 있다 |
| 환승 신호 | 이번 라운드에 환승역을 지나가거나 도착했다면 버스 1대를 지정해 신호 효과 적용 |

### 지하철 방만의 장점

- 충돌 위험이 없다.
- 환승역을 이용해 버스의 점수를 방해하거나 지원할 수 있다.
- 연속 이동 보너스로 꾸준히 점수를 얻기 쉽다.

### 지하철 방만의 단점

- 노선이 고정되어 있어 원하는 위치로 자유롭게 갈 수 없다.
- 목적지를 지나치면 다음 순환까지 기다려야 한다.
- 버스보다 직접적인 점수 폭발력은 낮다.

### 점수 획득 방식

| 조건 | 점수 |
| --- | ---: |
| 지정 역 도착 | +4 |
| 환승역 도착 | +3 |
| 한 라운드에 3역 이상 이동 | +2 |
| 환승 신호로 버스 감점 발생 | +2 |
| 원하는 지정 역을 지나치고 정차 실패 | -1 |

## 6. 방 사이 상호작용 규칙

3개의 방은 동시에 행동을 정하지만, 결과는 하나의 전체 맵에 합쳐진다. 운영자는 라운드 종료 후 아래 순서로 상호작용을 판단한다.

### 6.1 버스 근접 감점

라운드 종료 시 1번 버스와 2번 버스의 거리를 확인한다.

| 상태 | 처리 |
| --- | --- |
| 거리 2칸 이상 | 감점 없음 |
| 거리 1칸 | 두 버스 모두 `-2점` |
| 거리 1칸이고 2번 버스가 과속 상태 | 1번 버스 `-2점`, 2번 버스 `-3점` |

거리는 상하좌우 기준으로 계산한다. 대각선은 거리 2칸으로 보지 않고, 감점 없음으로 처리한다.

### 6.2 버스 충돌

아래 중 하나라도 해당하면 충돌이다.

- 두 버스가 같은 칸에 도착하려 했다.
- 한 버스가 다른 버스의 현재 칸으로 들어가려 했다.
- 두 버스의 이동 경로가 서로 교차했다.

충돌이 발생하면 두 버스 모두 `-6점`을 받는다. 충돌한 버스는 충돌 직전 칸에서 정지한 것으로 처리한다.

### 6.3 지하철 환승 신호

지하철이 이번 라운드에 S4, S8, S10 중 하나를 지나가거나 도착했다면, 지하철 방은 1번 버스 또는 2번 버스 중 하나를 지정해 신호를 줄 수 있다.

| 신호 종류 | 효과 |
| --- | --- |
| 지연 신호 | 지정한 버스가 이번 라운드 회색 공용 구역 보너스를 얻었다면 그 보너스를 취소한다 |
| 환승 지원 | 지정한 버스가 환승역 연결 칸 근처에서 라운드를 끝냈다면 해당 버스에 `+1점` |

한 라운드에 신호는 1개만 사용할 수 있다. 운영 난이도를 낮추기 위해 지연 신호와 환승 지원 중 하나만 선택한다.

### 6.4 중앙 회색 칸과 공용 구역

회색 공용 구역은 여러 방의 선택이 만나는 공간이다.

- 버스가 회색 칸에서 라운드를 끝내면 `+2점`
- 두 버스가 모두 회색 공용 구역 안에서 라운드를 끝내면 각자 `+1점`만 받는다.
- 두 버스가 회색 공용 구역 안에서 거리 1칸이면 회색 보너스를 받지 못하고 근접 감점을 적용한다.
- 지하철의 지연 신호는 회색 보너스를 취소할 수 있다.

### 6.5 방해와 도움

| 선택 | 다른 방에 주는 영향 |
| --- | --- |
| 1번 버스가 중앙을 선점 | 2번 버스의 중앙 보너스 진입 경로가 줄어든다 |
| 2번 버스가 과속으로 목적지 선점 | 1번 버스의 목적지 점수와 중앙 진입을 방해한다 |
| 지하철이 환승역을 통과 | 버스의 회색 보너스를 취소하거나 인접 버스를 지원한다 |
| 버스가 환승역 연결 칸 근처에 정차 | 지하철 환승 지원을 받을 수 있다 |

## 7. 점수 계산 방식

점수는 라운드마다 계산하고, 5라운드 종료 후 합산한다.

### 기본 점수표

| 점수 요소 | 조건 | 점수 |
| --- | --- | ---: |
| 목적지 도착 | 버스가 이번 라운드 목적지에 도착 | +5 |
| 색상 카드 또는 미션 성공 | 팀 색상, 개인 미션, 라운드 미션 달성 | +3 |
| 이동 효율 | 버스가 3칸 이하 이동으로 목적지 또는 미션 달성 | +1 |
| 충돌 회피 | 버스가 거리 2칸 이상을 유지하고 감점 없음 | +1 |
| 지하철 역 도착 | 지하철이 지정 역에 정확히 도착 | +4 |
| 지하철 환승역 도착 | S4, S8, S10에 도착 | +3 |
| 지하철 연속 이동 | 한 라운드에 3역 이상 이동 | +2 |
| 특수 칸 점령 | 버스가 회색 공용 구역에서 종료 | +2 |
| 라운드별 보너스 | 운영자가 공개한 라운드 보너스 달성 | +2 |
| 버스 근접 | 두 버스가 거리 1칸으로 종료 | -2 또는 -3 |
| 버스 충돌 | 같은 칸, 경로 교차, 상대 칸 진입 | -6 |
| 시간 초과 | 행동지 미제출 또는 늦은 제출 | -2 |
| 지하철 정차 실패 | 지정 역을 지나치고 정차하지 못함 | -1 |

1. 충돌 감점이 적은 팀/플레이어
2. 목적지 도착 횟수가 많은 팀/플레이어
3. 지하철 지정 역 도착 횟수가 많은 팀/플레이어
4. 그래도 같으면 공동 우승

## 8. 플레이어용 1페이지 요약 규칙

### 게임 목표

5라운드 동안 1번 버스, 2번 버스, 지하철을 조작해 가장 많은 점수를 얻는다.

### 내가 해야 할 일

- 라운드마다 배정된 방으로 이동한다.
- 방 안 사람들과 이번 라운드 행동을 정한다.
- 제한 시간 안에 행동지를 작성해 제출한다.
- 내 팀 또는 개인 점수를 올릴 선택을 한다.

### 방별 역할

| 방 | 할 일 |
| --- | --- |
| 1번 버스 | 안정적으로 목적지와 색상 미션을 노린다 |
| 2번 버스 | 빠르게 움직이고 1번 버스를 견제한다 |
| 지하철 | 정해진 노선을 따라 역 보너스와 환승 신호를 노린다 |

### 점수 얻는 법

- 목적지 도착: `+5`
- 미션 성공: `+3`
- 회색 공용 구역 점령: `+2`
- 지하철 지정 역 도착: `+4`
- 환승역 도착: `+3`
- 라운드 보너스 달성: `+2`

### 감점되는 행동

- 버스끼리 거리 1칸으로 끝남: `-2`
- 과속한 2번 버스가 거리 1칸으로 끝남: `-3`
- 버스 충돌: `-6`
- 행동지 시간 초과: `-2`
- 지하철 지정 역 정차 실패: `-1`

### 라운드 이동 방식

- 1라운드가 끝나면 운영자가 다음 방 배치를 알려준다.
- 내 P번호를 듣고 해당 방으로 이동한다.
- 방 이동 시간은 3분이다.

### 최종 승리 조건

5라운드 누적 점수가 가장 높은 팀 또는 플레이어가 우승한다.

## 9. 운영 난이도를 낮추기 위한 추천 수정안

1. 첫 진행에서는 개인전보다 5개 팀 2명제로 운영한다.
2. 전체 맵은 7 x 7 이하로 유지하고, 회색 공용 구역은 4칸만 둔다.
3. 지하철 신호는 한 라운드에 1번만 사용하게 한다.
4. 각 방 행동지는 객관식 체크형으로 만들어 작성 시간을 줄인다.
5. 점수 계산이 어렵다면 라운드별 보너스와 이동 효율 점수를 빼고 진행한다.
